# Converter temperatures and (Kelvin <-> Celsius) and currencies (TND <-> Euro).
## A small app that can be utilized as a converter .
##### To describe my project I use the CAR method (C : Chalenge , A : Action , R : Result)
###### C : So the challenge is to develop a small application that will serve as a converter, the application is capable of converting temperatures (Kelvin <-> Celsius) and currencies (TND <-> Euro) ...
###### A :I developed this app using Android. So I change some configuration in the manifest and I used java for the action and receive the information from the front app mobile, also I used xml to coding the design and I use the drag and drop method. In addition, I have included an Overflow options menu for navigating between pages.
###### R : The results left me satisfied and I sent this app to my teacher (This app serves as a basic homework) .
